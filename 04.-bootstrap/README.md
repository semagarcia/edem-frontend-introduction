# Ejercicio práctico 4

## Temática

Utilización de librerías CSS

## Objetivos

Entender el funcionamiento de una de las principales librerías CSS para el diseño responsivo

## Descripción

Hasta ahora hemos visto cómo poder dar un comportamiento responsivo a nuestro HTML, sin embargo, es muy
tedioso tener que manejar todo desde cero. Y si nos dedicamos al desarrollo web, habrá estilos y fragmentos
de código que serán siempre recurrentes. Todos sabemos qué puede ocurrir con el *copia&pega*, y la fuente de
errores que es a veces.

Por ello, existen librerías de CSS cuya misión es aglutinar esas clases CSS de ayuda para simplificar ese
proceso, facilitando el desarrollo e incrementando la productividad.

En este ejercicio veremos cómo poder incluir una de las principales y más conocidas: bootstrap. Para más
información se puede consultar [este enlace](https://getbootstrap.com/docs/5.1/getting-started/introduction/).

Para este ejercicio sólo será necesario crear una fila con varias columnas, y entender cómo utilizar las clases
que ofrece la librería.

## Ampliación

Como trabajo complementario se puede partir del ejercicio anterior para integrar Bootstrap.
